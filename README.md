📊 Expense Tracker System

A simple and efficient Expense Tracker System built using Python that helps users manage their daily expenses and keep track of their spending habits.


---

🚀 Features

➕ Add new expenses

📋 View all recorded expenses

💰 Track daily spending

🧾 Maintain financial records

⚡ Simple and user-friendly interface
